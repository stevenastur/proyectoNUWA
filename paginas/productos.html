PRODUCTOS

CALLERAS

A la hora de tener que tener que realizar actividades de colgado en la barra de dominadas, no pueden faltar las calleras para evitar dañarte las mano y proporcionar un mejor agarre.

MUÑEQUERA

A la hora de realizar cualquier tipo de actividad con las mano, siempre es conveniente utilizar muñequeras para evitar lesiones a futuro y molestia a la hora de trabajar.

SOGA PARA SALTAR

Con la soga se realizan actividades de todo tipo, pero es muy practico y eficas utilizar tu propia soga para tener siempre el mismo largo y sentir aun mas comodos los saltos o saltos dobles.

BARRA ENERGETICA

Siempre te puede salvar una barra energetica para utilizar de colacion pre o post entreno si es que venis apurado y con pocas horas para poder ingerir algo por tu parte.

RODILLERA

Las rodilleras son fundamnetal para aumentar la confianza y sentir nuestras rodillas mas fuertes a la hora de levantar pesos pesados.

GRIP DEDO

Los grip para los dedos son muy buenos para mejorar el agarre de la bara con el agarre gancho, tambien te protegen del rose con la misma barra y evita lesiones molestas.
